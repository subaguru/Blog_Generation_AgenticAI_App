# ✍️ Blog Generation Agentic AI App

An AI-powered application that autonomously generates full-length, SEO-optimized blog posts using **Agentic AI architecture**. Built using **LangChain**, **LLMs**, and external **tools** for research, fact-checking, and content formatting.

---

## 🚀 Features

- 🧠 Uses LangChain **agents** to reason and plan
- 🔍 Integrates **search tools** for live information gathering
- 📄 Generates structured **blog content** (intro, body, conclusion)
- 🖥️ Simple **Streamlit UI** for user interaction
